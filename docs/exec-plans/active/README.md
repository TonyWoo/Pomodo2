# Active Plans

Store one markdown file per non-trivial in-flight task in this directory.

## Rules

- Name files `<ticket>-<short-slug>.md`.
- Keep the sections for scope, assumptions, steps, validation, and progress
  log current while the work is active.
- Record the latest evidence needed for another agent to resume without chat
  history.
- Move the file to [../completed/](../completed/) after the work lands or is
  otherwise finished.
